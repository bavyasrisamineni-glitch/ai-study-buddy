# AI Study Buddy

## About the Project

AI Study Buddy is a simple AI-powered assistant built to help students study more effectively. Many students find it difficult to quickly understand new topics, revise before exams, or generate practice questions. This project aims to make studying easier by providing instant summaries, explanations, and quizzes using AI.

---

## What This Project Can Do

- Generate short and clear summaries of any topic  
- Create practice questions (MCQs)  
- Answer basic conceptual doubts  
- Provide an interactive chat-based study experience  

---

## Technologies Used

- Python  
- Streamlit (for the user interface)  
- OpenAI API / LLM API  
- Basic NLP concepts  

---

## How It Works

1. The user enters a topic or question.  
2. The system sends the request to an AI model using an API.  
3. The AI generates a response (summary, quiz, or explanation).  
4. The response is displayed in a simple web interface built using Streamlit.
